# URL Shortener

This is a URL shortening project built using the MERN stack (MongoDB, Express, React, Node.js). The project allows users to reduce the lenght of lengthy URL's.

## Getting Started

These instructions will help you set up the project on your local machine for development and testing purposes.

### Prerequisites

- Node.js and npm installed on your machine

### Installation

1. Fork the repository on GitHub.
2. Clone the repository to your local machine.
3. Navigate to the server directory and create a new file named `.env`.
4. Add your Mongodb cluster link to the `.env` file.
5. In the server directory, run `npm install` to install the necessary dependencies.
6. Start the server by running `node server.js`.
7. Navigate to the client directory and run `npm install` to install the necessary dependencies.
8. Start the client by running `npm run dev`.

## Usage

Once the project is set up and running, you can compress the lengthy Url to short URL .
